# Bayesian Probability Project

## Overview
This project focuses on **Bayesian probability** and **Bayes’ theorem**.  
You will learn how to compute likelihoods, priors, posteriors, and understand how Bayesian inference works in practical examples, such as medical studies.

---

## Learning Objectives
By the end of this project, you should be able to:

- Explain what **Bayesian probability** is.
- Use **Bayes’ theorem** to compute posterior probabilities.
- Understand the meaning of **prior**, **posterior**, **likelihood**, and **marginal probability**.
- Apply these concepts to real-world problems, e.g., drug side effects.
