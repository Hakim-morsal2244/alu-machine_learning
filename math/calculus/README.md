# Calculus

This project covers fundamental calculus concepts used in machine learning and mathematics.

## Learning Objectives

At the end of this project, I should be able to explain:

- Summation and product notation
- Series and common mathematical series
- Derivatives and derivative rules (product rule, chain rule)
- Partial derivatives
- Indefinite and definite integrals
- Double integrals

## Requirements

- Ubuntu 16.04 LTS
- Python3 (version 3.5)
- All files must be executable
- All Python files must follow pycodestyle (version 2.5)
- No external modules unless otherwise specified
