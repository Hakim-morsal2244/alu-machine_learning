# Convolutions and Pooling in Python

## Project Overview
This project implements **convolution and pooling operations** on grayscale images using Python and NumPy.  
The main focus is to understand how image feature extraction works in deep learning, including:

- Valid convolution
- Same convolution
- Max pooling
- Average pooling

These operations are implemented from scratch **without using built-in convolution functions**.
