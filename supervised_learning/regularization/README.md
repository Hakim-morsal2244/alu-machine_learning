# Regularization

## Description

This project covers regularization techniques used in machine learning and deep learning to reduce overfitting and improve model generalization.

## Learning Objectives

By completing this project, you should understand:

* What regularization is and why it is important
* L1 and L2 regularization
* Dropout regularization
* Early stopping
* Data augmentation
* How to implement regularization techniques using NumPy and TensorFlow
* Advantages and disadvantages of different regularization methods

## Requirements

* Ubuntu 16.04 LTS
* Python 3.5
* NumPy 1.15
* TensorFlow
* pycodestyle 2.4

## Tasks

### 0. L2 Regularization Cost

Calculate the cost of a neural network with L2 regularization.

### 1. L2 Gradient Descent

Implement gradient descent with L2 regularization.

### 2. L2 Regularization Cost in TensorFlow

Create a neural network using L2 regularization in TensorFlow.

### 3. L2 Regularization Layer

Implement an L2 regularized layer.

### 4. Dropout Forward Propagation

Implement forward propagation using dropout.

### 5. Dropout Gradient Descent

Implement gradient descent with dropout.

### 6. Dropout in TensorFlow

Create dropout layers in TensorFlow.

### 7. Early Stopping

Implement early stopping for neural network training.

### 8. Deep Neural Network with Regularization

Combine multiple regularization techniques.
