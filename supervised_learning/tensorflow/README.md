# ALU Machine Learning - TensorFlow Project

This project contains implementations of basic neural network components using TensorFlow 1.x style (compatibility mode in TensorFlow 2).

## Requirements

- Python 3.5 / 3.6 (ALU environment)
- TensorFlow 1.12
- NumPy 1.15
- Ubuntu 16.04 LTS (or WSL equivalent)
