# Databases

This project covers relational and non-relational databases.

## Concepts Covered

### MySQL
- Creating databases
- Creating tables with constraints
- SQL queries
- Indexes
- Stored procedures
- Functions
- Views
- Triggers
- ACID properties

### NoSQL / MongoDB
- Document storage
- MongoDB queries
- Insert, update, and delete operations
- Aggregation

## Requirements

- MySQL 5.7 compatible SQL scripts
- MongoDB 4.2 compatible files
- Python 3 with PyMongo
